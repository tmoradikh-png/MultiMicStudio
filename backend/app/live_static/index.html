MultiMic

Choose what you want to do.

Mode

📼 Record & Process
Phones record, upload, and get a synced studio mix to play and download. Best quality. Use the mobile app or main dashboard.
🎙️ Live Speaker — Be a microphone
Use this phone as a live wireless mic. Enter a room code and go live. Pick Left/Right for stereo.
🔊 Live Speaker — Play the sound
Open on the device connected to the speaker (PC, host phone, Bluetooth speaker). Same room code connects them.

How are you connecting?

All of these work — the audio travels through the server, so phones don't need to be on the same network.

📶 Same Wi-Fi
Best latency. Phone and speaker on the same network.
📱 Mobile Internet
Supported. Phone on 4G/5G, speaker anywhere online.
🔥 Hotspot
Supported. Share one phone's data to the others.
🔵 Bluetooth Speaker
Pair the speaker to the listener device, then press Start.
Order to try: one phone mic → speaker on Wi-Fi, then over mobile data, then two phones for Left/Right stereo, then several phones for party/room mode.
